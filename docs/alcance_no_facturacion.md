# Alcance: no facturacion comercial

Este proyecto no emite facturas electronicas de venta.

Este proyecto no emite boletas de venta.

Este proyecto no emite notas de credito comerciales.

Este proyecto no emite notas de debito comerciales.

Este proyecto no emite guias de remision.

Este proyecto no calcula IGV de ventas.

Este proyecto no envia comprobantes comerciales a SUNAT.

El sistema si genera boletas de pago de remuneraciones.

El sistema si genera lotes de pago de sueldos.

El sistema si genera reportes de costo laboral por edificio.

Cualquier facturacion al cliente, comite o edificio se hace fuera del sistema, en un facturador externo.

Integracion futura permitida: Exportacion para facturador externo: CSV/XLSX/JSON con resumen de costo laboral por edificio, sin emision de comprobantes comerciales.
